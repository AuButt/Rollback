GGPO Style Netcode.

A simple 2d fighter made with Rollback netcode over delay-based using GGPO architecture.

While initially going into this project, I wanted to implement the existing GGPO middleware into Unity and build the game off of existing systems. I wanted to take a swing at developing it from the ground up!
